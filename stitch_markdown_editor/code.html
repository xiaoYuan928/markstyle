# The Art of Typography
Typography is the **art and technique** of arranging type to make written language legible, readable, and appealing when displayed.
## Key Elements
* **Typeface**: A family of fonts (e.g., Helvetica).
* **Hierarchy**: Using size and weight to guide the reader.
* **Leading**: The vertical space between lines of text.
> "Typography is the voice of the written word."
Effective typography manages the balance between whitespace and content.
```css
body {
font-family: 'Inter', sans-serif;
line-height: 1.6;
}
```
